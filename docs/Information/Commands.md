# Commands

These are all of the commands you can do

--------------------------------------------------------------------------------------------------

### TPA
- /tpa [PLAYER NAME]
TPA's to a player.
- /tpaccept
Accept's a TPA Request.
- /tpadeny
Denies a TPA Request.
- /tpahere [PLAYER NAME]
Sends a TPA Request to that player for them to come to you.
